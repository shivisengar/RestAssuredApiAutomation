# RestAssured API Automation Framework

## Overview
This project is an API automation framework built using Rest Assured, TestNG, Java, Maven, and Excel for data-driven testing.

## Tech Stack
- Java
- Rest Assured
- TestNG
- Maven
- Apache POI (Excel)
- GitHub

## Project Structure
- src/main/java  
  - endpoints (Routes)
  - services (API calls)
  - utils (Config, Request Spec, Excel Utils)
- src/test/java  
  - test classes
- src/test/resources  
  - test data (Excel files)
  - configuration files

## Prerequisites
- Java 8+
- Maven
- Git
